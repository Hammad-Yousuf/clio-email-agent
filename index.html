Clio Matter Mapping Agent

Azure Functions-based email classification and writeback system for Clio Manage

Service Operational

API Endpoints

POST /api/classify Classification only
POST /api/classify-and-writeback Classify + auto writeback
POST /api/writeback Force communication writeback
GET /api/matters Cached matters snapshot
GET /api/health Health check

Features

  • Email Classification - Matches emails to matters using multiple signals
  • Confidence Scoring - Weighted scoring with configurable threshold
  • Automatic Writeback - High confidence → Communication, Low confidence → Draft Note
  • Audit Logging - All operations logged to Azure Table Storage
  • Matter Caching - Local cache for improved performance
